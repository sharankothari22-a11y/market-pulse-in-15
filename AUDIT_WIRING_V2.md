## Top 3 Surprises Since V1

1. **Macro Live Indices is wireable in 45 min** — V1 called this a DB stub. It is, but `MacroDashboard.jsx` already knows the exact yfinance symbols it needs (`^NSEI`, `^BSESN`, `^GSPC`, `^DJI`, `^IXIC`, `^VIX`). The backend already has `fetch_indices_safe()` for two of them. Adding a yfinance fallback to `/api/macro` would light up the entire Live Indices section with zero frontend changes.

2. **L07 `factor_deltas` are silently computed but have no UI home** — Unlike the stub items that return empty data, factor_deltas are ACTUALLY computed every analyze run (signals → assumption deltas), included in the JSON response, but no panel reads them. This is a "full pipeline, no display" gap — the highest-quality data that isn't shown anywhere.

3. **Chat context wiring is a one-sided backend gap** — The frontend already sends `session_id` to `POST /api/chat/message`. The backend receives it in the request body but the handler reads only `message`. One `body.get("session_id")` + `_find_rp_session()` call + a ticker/scoring context string injected into the system prompt is all that's needed.
