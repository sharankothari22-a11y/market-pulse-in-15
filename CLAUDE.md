# Market Pulse — Project Context

Indian equity research platform. Backend: FastAPI (`server.py`) + Jupyter notebooks (papermill). Frontend: React (`frontend/`). Notebooks live under `notebooks/`. Core engine under `research_platform/`.

## Scope discipline
This project is in maintenance mode, not redesign mode. Do exactly what
is asked. If you spot something that "should" be fixed — flag it in the
reply, do not fix it. The user will scope work explicitly.
